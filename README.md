# Retail Sales Performance Analysis

This project analyses retail sales data to uncover insights on revenue, profitability, and regional performance.

## Tools Used
- SQL (PostgreSQL)
- Tableau

## Key Analysis
- Sales and profit by category
- Regional performance comparison
- Monthly sales trend analysis

## Key Insights
- Technology drives highest revenue and profit
- Furniture has low profitability due to higher discounting
- Strong seasonal sales trend in Q4
- Central region shows lower profit efficiency
